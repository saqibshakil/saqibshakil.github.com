Authenticated Modules app.GL.ModuleRouter

In most applications there is a requirement for a module to only be accesible if the user is authenticated. Else the user is redirected to the login view and directed back to the requested route. implementation of this behavior simply requires you to extend the Router from app.GL.ModuleRouter class instead of Backbone.SubRoute

Changes

The only mentionable change is that instead of before being called before each route onBefore is called
